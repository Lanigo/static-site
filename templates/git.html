Git and Github

What is Git? When you’re dealing with more than one person working on something, or on a large project with lots of complicated components you need version control
What is Github?A Web-based Git repository hosting service, which offers all of the distributed revision control and source code management (SCM) functionality of Git as well as adding its own features.
If you plan on taking web development seriously, you need to have a good understanding of both.
